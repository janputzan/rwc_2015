Response:

I have implemented application against the provisional designs. The screen of the application is divided into three parts: side menu, main content and collapsible panel where details are shown. The cards mentioned in the feedback are displayed in the main container depending on the user action. The content of the carsd is dynamically pulled from the database using AJAX and appended to the document using jQuery. Clicking on the card will show specific information in detail on the collapsible part.

Features
  • Gallery slider - a plugin I wrote using jQuery
  • Collapsible Panel - on the right hand side
  • Information pulled from the database using AJAX and appended with jQuery
  • Graphical representation of the data pulled from the databse in the form of chart graphs
  • jQuery events used: mouseover, click
  • jQuery plugin - slideMe.js
Materials used
  • Materialize CSS - CSS framework - I used some of its features to style the page
  • jQuery - Core functionality of the applications
  • jQuery UI - Additional functionality of the application (effects)
  • Chart.js - I used it for statistics page - showing graph charts
